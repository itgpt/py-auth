# Py-Auth-ITGPT 快速开始指南

## 🚀 立即开始

### 1. 进入项目目录
```bash
cd ~/HermesWork/projects/py-auth-itgpt
```

### 2. 查看可用命令
```bash
make help
```

### 3. 安装开发环境（推荐）
```bash
# 创建虚拟环境（可选）
python -m venv venv
source venv/bin/activate

# 安装开发依赖
make dev-install

# 安装 pre-commit 钩子
make pre-commit
```

### 4. 运行代码质量检查
```bash
make check
```

### 5. 启动服务
```bash
# 方式1：直接运行
python main.py

# 方式2：使用 Docker
docker-compose up
```

## 🆘 常见问题

### 1. 依赖安装失败
```bash
# 使用系统包管理器
sudo apt update
sudo apt install python3-venv python3-pip

# 或使用 pipx
pipx install pip
```

### 2. Docker 启动失败
```bash
# 检查 Docker 服务
sudo systemctl status docker

# 重建镜像
docker-compose build --no-cache
```

### 3. 代码检查报错
```bash
# 自动修复格式
make format

# 自动修复导入排序
isort .
```
